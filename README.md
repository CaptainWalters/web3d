# Installation guide

### Prerequesits

```
Node,
npm,
php >=5.6
```

### Install

Clone repo  
`git clone`

Install dependencies  
`npm install`

Run server  
`gulp serve`

Gulp will then build and serve the website at `localhost:3000/index.php`

The source code is located in `/src` and the site will be built to `/build`
